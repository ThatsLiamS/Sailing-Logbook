## Development

#### **Install locally**
```bash
$ git clone https://github.com/ThatsLiamS/Sailing-Logbook
$ npm ci
```

#### **Complile with a hot-reload for development**
```bash
$ npm run dev
```

#### **Complile, Minify along with a preview for production**
```bash
$ npm run build && npm run preview
```
